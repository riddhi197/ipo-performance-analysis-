# IPO Performance Analysis and Listing Gain Prediction

## Overview

This project analyzes IPO performance using historical IPO data and machine learning techniques.

The objective is to identify factors affecting IPO listing gains and build predictive models using subscription and pricing data.

## Dataset

* Total IPOs Analyzed: 561
* Features:

  * Issue Size
  * QIB Subscription
  * HNI Subscription
  * RII Subscription
  * Total Subscription
  * Offer Price
  * Listing Price
  * Listing Gain
  * Current Gains

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

## Analysis Performed

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Correlation Analysis
* Visualization
* Linear Regression
* Random Forest Regression
* Model Evaluation

## Key Findings

* Total Subscription showed the strongest relationship with listing gains.
* QIB and HNI participation were important indicators of IPO success.
* Investor demand had greater impact than issue size.

## Machine Learning Models

1. Linear Regression
2. Random Forest Regressor
